# 전화

카카오톡으로 상대방에게 "전화가능?" 이라고 물을 때 올바른 영어 표현으로는 무엇이 있을까?

- Can I talk to you for a minute now?
- Would you mind if I talk to you over the phone now?
- Is it find to talk to you now?
- Can you talk now?
